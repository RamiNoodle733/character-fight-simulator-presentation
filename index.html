Character Fight Simulator

iOS Shortcuts are typically used for small, everyday tasks like playing the most recent album, retrieving location data from an image, or quickly accessing a specific app. However, I wanted to push the boundaries of what’s possible within the iOS Shortcuts framework. The Character Fight Simulator is an ambitious project that transforms this simple automation tool into a dynamic, interactive program. It generates fight animations between any two user-selected characters over a chosen background, all without using any input files. Instead, it retrieves all relevant images and quotes from the web in real-time, and includes features such as randomized attack power and order, various effects, character quotes, and a winner determined by healthbar depletion. This project demonstrates how iOS Shortcuts can be utilized not just for minor tasks, but for creating complex, entertaining experiences.
